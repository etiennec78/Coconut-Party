# Coconut partY 🏝️
This game is a Tower Defense game where crabs try to attack the Monkey King. The goal is to place monkeys along the path to defend the Monkey King for as long as possible. _(This is a school project)_

## Compilation ⚙️

```sh
make
```

## Execution 🚀

```sh
./game
```

## Cleaning 🧹
```sh
make clean
```
